# 后端需求
1. 用户 登录 && 注册
2. 用户 添加 角色
3. 用户 删除 角色
4. 用户 查询 角色
5. 用户与角色文字聊天
6. 用户与给角色发送语音
7. 用户与角色语音通话聊天

# API 接口
用户注册（user/register）POST
入参：
username 用户名
password 密码
出参：网络码 200

用户登录 （user/login）POST
入参：
username 用户名
password 密码
出参：JWT 


用户添加角色 （user/addRole）POST
入参：
JWT
角色名称
角色描述（存入知识库）

出参：网络码 200 + 角色 ID
